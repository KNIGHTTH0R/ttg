Scenario Victory Points

This group of keys define how the game is scored. Each of these settings is an equasion that is used to calculate victory points at certain times. Most mathematical expressions are accepted. Variables can be used in these expressions pertaining either to worlds or ships. Different variables are available for each time and are contained in tables at the end of this page. There are two general reccomended modes of distributing victory points. Both can be used to achieve the same ends, but in different ways. Or combined in some interesting method of calucation.

Mode one: posession. Set the values of either or both VPHave keys. Victory Points are then measured based on what someone currently has or posesses.

Mode two: aquisition. Set the values of either or both VPGain and VPLose keys. Victory Points are then measured based on what someone gains or loses.

VPHaveWorld This specifies the victory points for having a world. When a world changes ownership this value is removed from the lower and awarded to the winner. It should only appear once.

VPGainWorld This specifies the victory points for gaining a world. When a world changes ownership this value is awarded to the winner. It should only appear once.

VPLoseWorld This specifies the victory points for losing a world. When a world changes ownership this value is removed from the loser. It should only appear once.

VPHaveShip This specifies the victory points for having a ship. They are assessed at the begining of the game. Each time a ship is lost this value is removed from the loser. It should only appear once.

VPGainShip This specifies the victory points for gaining a ship. If the game supports ship construction, this value is awarded anyone who makes a ship. It should only appear once.

VPLoseShip This specifies the victory points for losing a ship. When a ship is destroyed in combat, this value is removed from the loser. It should only appear once.
